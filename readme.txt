Proyecto: AgrazStore (Prototipo)
Desarrollado en: NetBeans IDE
Lenguaje: Java
Tipo de proyecto: Aplicación de escritorio (Swing)

----------------------------------------
REQUISITOS DEL SISTEMA
----------------------------------------
- JDK 8 o superior
- NetBeans IDE (recomendado: versión 12.6 o superior)
- Sistema operativo: Windows, Linux o macOS
- Librería GSON (versión 2.10.1)

----------------------------------------
INSTRUCCIONES DE INSTALACIÓN Y USO
----------------------------------------

1. Descarga e instala el JDK desde:
   https://www.oracle.com/java/technologies/javase-downloads.html

2. Instala NetBeans desde:
   https://netbeans.apache.org/download/index.html

3. Descarga la librería GSON (gson-2.10.1.jar) desde:
   https://repo1.maven.org/maven2/com/google/code/gson/gson/2.10.1/gson-2.10.1.jar
   (O utiliza el archivo adjunto en la entrega si está incluido)

4. Abre NetBeans y selecciona:
   "File/Archivo" > "Abrir Proyecto"

5. Navega hasta la carpeta descomprimida de este archivo ZIP
   y selecciona el proyecto "AgrazStore Digital".

6. Espera a que NetBeans cargue el proyecto completamente.

7. Para agregar la librería GSON:
   - Haz click derecho sobre el proyecto
   - Selecciona "Properties/Propiedades"
   - En el panel izquierdo, selecciona "Libraries/Bibliotecas"
   - Haz click en "Classpath" y luego en el botón "Add JAR/Folder..." o "+"
   - Busca y selecciona el archivo "gson-2.10.1.jar" que descargaste
   - Haz click en "OK" para confirmar y cierra la ventana de propiedades

8. Para ejecutar:
   - Haz clic derecho en el proyecto > "Ejecutar/Run"
   - O dale a la flecha verde en el panel superior

9. Al compilar observando la ventana de iniciar sesión:
   - Todo en minúscula
   - Para ver la interfaz de Cliente, ingresa: 
     usuario: cliente  ; contraseña: 1234
   - Para ver la interfaz de Empleado, ingresa: 
     usuario: empleado  ; contraseña: 1234

10. En ambas ventanas se encuentra un boton con el simbolo '?'
    al darle click, este despliega un cuadro de dialogo con 
    las instrucciones de cada página
